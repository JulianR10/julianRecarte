## 📝 Próximos pasos después de instalar:

1. **npm run dev** - Inicia servidor en localhost:3000
2. Abre `http://localhost:3000` en tu navegador
3. ¡Comienza a editar archivos y verás los cambios en tiempo real!

### Crear nuevos componentes:
```
src/components/MiComponente.astro
```

### Crear nuevas páginas:
```
src/pages/nueva-pagina.astro
```

### Agregar animaciones GSAP:
```javascript
import gsap from 'gsap';
// Tu código de animación aquí
```

---

## 💡 Tips importantes:

- **Edita** archivos `.astro` directamente
- **Estilos** usa TailwindCSS con clases (más rápido que CSS manual)
- **Animaciones** usa GSAP para interactividad avanzada
- **Deploy** en Vercel con un click (está optimizado)
