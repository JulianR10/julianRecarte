# Cómo empezar

## Requisitos
- Node.js 18+ instalado
- npm o pnpm

## Pasos de instalación

1. Abre una terminal en la carpeta del proyecto
2. Ejecuta:
```bash
npm install
```

3. Una vez instalado, inicia el servidor de desarrollo:
```bash
npm run dev
```

4. Abre tu navegador en `http://localhost:3000`

## Estructura lista para usar

✅ Layout base configurado con Lenis para scroll suave
✅ Tailwind CSS integrado
✅ GSAP listo para animaciones
✅ Componentes base (Header, Button)
✅ Estilos globales con tipografía personalizada
✅ Pages estructurado correctamente

## Próximas mejoras

Una vez instales y ejecutes `npm run dev`, puedes:
- Crear nuevos componentes en `src/components/`
- Añadir más páginas en `src/pages/`
- Usar GSAP para animaciones avanzadas
- Personalizar diseño en `tailwind.config.mjs`

¡Listo para crear experiencias web interactivas! 🚀
